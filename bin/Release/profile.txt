// Command List

// windowsVersion          - Windows version to simulate for enhanced pointer precision.
//                           Possible values: XP, Vista, 7

//

//                           Default: 1.0
//

//                           Possible values: 0 (none), 1 (enhanced pointer precision), 2 (non-interpolated), 3 (linear)
//                           Default: 1
//
// subPixelation           - Whether or not sub-pixelation is forced for XP and Vista style enhanced pointer precision. Keeping this on is recommended.
//                           Possible values: 0 (off), 1 (on)
//                           Default: 1
//
// screenResolution        - The resolution of the screen, in PPI. Used with Windows accel.
//                           Possible values: Accepts any valid unsigned integer
//                           Default: 96
//
// screenRefreshRate       - The refresh rate of the screen, in hertz. Used only with Windows 7 accel.
//                           Possible values: Accepts any valid unsigned integer
//                           Default: 60
//
// thresholdsX             - An array of thresholds for the speed of the mouse.
//                         - Possible values: An array of 5 floating point numbers, delimited by spaces
//                         - Default: 0.0 0.43 1.25 3.86 40.0
//
// thresholdsY             - An array of thresholds for the resulting speed of the pointer.
//                         - Possible values: An array of 5 floating point numbers, delimited by spaces
//                         - Default: 0.0 1.37 5.30 24.30 568.0
//
// verbose                 - Enables a verbose output mode on the terminal.
//                         - Possible values: Doesn't take in a value.
//                         - Default: Disabled.
